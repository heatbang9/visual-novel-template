# 프로젝트 개요: 시계탑의 메아리

<logline>언덕 위 시계탑에서 울리는 종소리가 과거와 현재를 잇는다. 주인공은 시간 지연 편지 "메아리 우편"을 통해 20년 전 익사 사건과 현재의 실종을 연결하며, 죄와 용서의 균형을 찾는다.</logline>

<theme>책임과 용서, 시간의 왜곡, 침묵과 고백</theme>

<setting>
- 장소: 해안 도시 카데나, 언덕 위 녹슨 시계탑, 폐쇄된 구 우체국, 파도 부딪히는 절벽길
- 시간: 늦여름 5일 밤, 매일 자정 종이 한 번씩 울림
- 질감: 스팀펑크 잔해 + 고딕 해안 노이르
</setting>

<cast>
- 주인공: 윤시안 (17, 고2) — 태엽을 만지면 지난 시간을 듣는 촉각 공감
- 동료: 강해진 (18, 드론 동호회), 류미연 (17, 문예부 편집장)
- 의혹 대상: 도리안 사제 (시계탑 관리), 한소정 (해안 경찰), 정무열 (과거 피해자 형)
- 미지의 발신자: "H" — 모든 메아리 우편의 원발신자로 추정
</cast>

<mbti>
- 윤시안: INFP | 촉각 공감 | 고소공포 | 잔향 기록癖
- 강해진: ENTP | 기계 광 | 충동적 | 드론 해킹
- 류미연: INTJ | 문서 탐사 | 감정 절제 | 숨겨진 죄책감
- 도리안: INFJ | 신앙과 비밀 | 죄의식 | 종 관리
- 한소정: ESTP | 현장파 | 실적 압박 | 바다 공포
</mbti>

## 플레이 토너
- 4막 + 프롤로그/에필로그
- 매 밤 자정 종소리 직후, 과거/미래 편지 교환 장면 존재
- 기억 상태 흐름: clarity → hum(잔향) → echo(반향) → distort(왜곡) → fixed(복원)

## 시스템 메모
- `<letter target_time="past|future">`를 XML `<item>`이나 `<document>`로 매핑
- 종 패턴(3-5-3-1-2) 입력 퍼즐 → deduction 체크
- 시계탑 외벽 클라임 시 resolve 체크, 실패 시 기억 상태가 distort로 하락
