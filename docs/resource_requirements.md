## 권장 스펙

### 캐릭터 스프라이트
- **해상도**: 512x1024px 또는 1024x2048px
- **포맷**: PNG (투명 배경)
- **스타일**: 일관된 아트 스타일
- **위치**: 캐릭터는 중앙 하단에 정렬

### 배경 이미지
- **해상도**: 1920x1080px (16:9 비율)
- **포맷**: PNG 또는 JPG
- **스타일**: 캐릭터와 조화를 이루는 스타일

## 구현 상태

현재 모든 리소스 파일들은 빈 placeholder 파일로 생성되어 있습니다. 
실제 아트워크로 교체하여 게임을 완성할 수 있습니다.

### 자동 생성된 placeholder 파일들
```
characters/
├── student_player/
│   ├── normal.png
│   ├── happy.png
│   ├── worried.png
│   └── surprised.png
├── friend_mina/
│   ├── normal.png
│   ├── cheerful.png
│   ├── excited.png
│   └── encouraging.png
...

backgrounds/
├── school_hallway.png
├── school_classroom.png
├── magic_academy_entrance.png
└── ...
```

## 향후 확장

새로운 에피소드나 캐릭터를 추가할 때는:

1. `scenarios/{episode_name}/characters/` 에 캐릭터 JSON 파일 추가
2. `characters/{character_id}/` 디렉토리에 감정별 스프라이트 추가
3. XML 시나리오에서 참조하는 배경을 `backgrounds/` 에 추가
4. 이 문서 업데이트
